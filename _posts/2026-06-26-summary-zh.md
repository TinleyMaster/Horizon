## [llama.cpp b9788 新增 SYCL 张量并行与双 GPU 支持](https://github.com/ggml-org/llama.cpp/releases/tag/b9788) ⭐️ 7.0/10

llama.cpp 版本 b9788 引入了 SYCL 张量并行，支持双 GPU 全规约，并包含 BF16 压缩以实现高效的大张量通信。 该功能显著提升了 Intel GPU 上的 LLM 推理性能，使双 GPU 配置相比层模式拆分，令牌生成速度最高提升 78.6%。 该实现针对双 GPU 使用退化的环全规约，以 32768 个元素为阈值，在 FP32 直接路径和 BF16 压缩路径之间切换，对大张量将 PCIe 流量减半。

github · github-actions[bot] · 6月25日 08:45

**背景**: llama.cpp 是一个流行的开源项目，用于在消费级硬件上本地运行大语言模型（LLM）。张量并行将模型层拆分到多个 GPU 上，以减少每个设备的内存占用并加速推理。SYCL 是一种用于异构计算的跨平台编程模型，支持 Intel GPU。
