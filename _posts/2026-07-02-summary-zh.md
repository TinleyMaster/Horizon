## [[科技] 首个能生长分裂的合成细胞诞生](https://www.quantamagazine.org/for-the-first-time-a-cell-built-from-scratch-grows-and-divides-20260701/) ⭐️ 10.0/10

**发生了什么**: 由 Kate Adamala 博士领导的研究人员创造了 SpudCell，这是首个从零构建并能生长和分裂的合成细胞。这一突破于 2026 年 7 月公布，克服了合成生物学中的一个重大障碍。

**投资影响**: 这一成就标志着向从零构建生命迈出了重要一步，在生物技术、医学和理解生命起源方面具有潜在应用。它证明了最小的合成细胞可以在没有细胞骨架的情况下执行分裂等复杂任务。

**关键信息**: SpudCell 使用了一种不依赖细胞骨架的独特分裂机制，简化了过程。该研究最初被《细胞》期刊拒绝，但已广泛分享，手稿可在 Biotic.org 上获取。

hackernews · defrost · 7月1日 14:20 · [社区讨论](https://news.ycombinator.com/item?id=48747304)

**背景**: 合成生物学旨在从非生命成分创造人工细胞。之前的努力实现了喂养、生长和 DNA 复制，但细胞分裂仍然难以实现，因为它通常需要复杂的细胞骨架。Adamala 的团队通过设计更简单的分裂过程绕过了这一难题。

**社区讨论**: 评论强调了这一突破的重要性，一些人注意到论文被拒以及作者非常规的沟通策略引发的争议。一条推测性评论想象了 Biotic 成为主导实体的未来，反映了合成生命的深远影响。

**发生了什么**: Cloudflare 推出了一个货币化网关，允许网站运营商通过 x402 协议对 Cloudflare 背后的任何资源收费，从而通过加密货币支付实现 API 访问的微交易。

**投资影响**: 这项创新可能从根本上改变 API 和网络资源的货币化方式，实现无需传统支付轨道的无缝微交易，并可能通过要求付费访问来减少机器人流量。

**关键信息**: 该网关使用 HTTP 402 Payment Required 状态码并支持加密货币支付，允许运营商为任何资源设置自定义定价。它旨在与 Cloudflare 的现有基础设施无缝协作。

hackernews · soheilpro · 7月1日 13:59 · [社区讨论](https://news.ycombinator.com/item?id=48746914)

**背景**: HTTP 402 状态码早已定义但很少被实现。Cloudflare 的网关利用此代码创建了一种标准化的方式，在提供内容之前请求付款，从而可能解决 API 访问的货币化问题并减少来自机器人的滥用。

**社区讨论**: 社区评论既表达了兴奋也表达了怀疑。一些人认为这是微交易和基于代理的支付的梦想，而另一些人则担心它可能催生新的垃圾邮件形式以及围绕发票和税收的法律复杂性。
