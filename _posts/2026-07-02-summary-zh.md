## [[科技] 首个能生长分裂的合成细胞诞生](https://www.quantamagazine.org/for-the-first-time-a-cell-built-from-scratch-grows-and-divides-20260701/) ⭐️ 10.0/10

由 Kate Adamala 博士领导的研究人员创造了 SpudCell，这是首个从零构建并能生长和分裂的合成细胞。这一突破克服了合成生物学中长期存在的障碍。 这一成就标志着从零构建生命的范式转变，在生物技术、医学和材料科学等领域具有潜在应用。它表明，无需天然细胞骨架即可工程化实现分裂等复杂细胞行为。 SpudCell 通过摒弃细胞骨架、采用不同机制实现分裂，从而简化了过程。该工作最初被《细胞》期刊拒绝，一些批评者认为 SpudCell 并非真正的生物学。

hackernews · defrost · 7月1日 14:20 · [社区讨论](https://news.ycombinator.com/item?id=48747304)

**背景**: 合成生物学旨在从分子组分创建人工细胞。以往的努力可以喂养合成细胞并复制 DNA，但细胞分裂一直难以实现，因为它通常需要复杂的细胞骨架重组。Adamala 团队通过设计不依赖细胞骨架的分裂机制绕过了这一难题。

**社区讨论**: 社区参与度很高，获得 825 分和 271 条评论。一些评论者注意到该工作被《细胞》拒绝以及作者非常规的预印本策略引发的争议。其他人则推测未来影响，包括一个虚构场景：合成细胞公司 Biotic 成为主导政体。

## [[科技] Cloudflare 货币化网关通过 x402 实现微交易](https://blog.cloudflare.com/monetization-gateway/) ⭐️ 8.0/10

Cloudflare 推出了一款货币化网关，允许网站运营者通过 HTTP 402 状态码对 Cloudflare 背后的任何资源收费，从而实现对网页内容和 API 访问的微交易。 这可能从根本上改变网络内容和 API 的货币化方式，实现无需传统支付基础设施的按使用付费模式，并可能减少对广告或订阅的依赖。 该网关使用 x402 协议，扩展了 HTTP 402（需要付款）以促进实时微交易。它与 Cloudflare 现有的安全和性能功能集成，并支持包括加密货币在内的多种支付方式。

hackernews · soheilpro · 7月1日 13:59 · [社区讨论](https://news.ycombinator.com/item?id=48746914)

**背景**: HTTP 402（需要付款）是 HTTP 规范中定义的一个很少使用的状态码。Cloudflare 的实现使其重新用于实际用途，支持新的货币化层，可以支持代理间支付，并减少机器人驱动的成本上涨对网站运营者的影响。

**社区讨论**: 社区评论既表达了兴奋也表达了怀疑。一些人认为这是期待已久的基于代理的微交易解决方案，而另一些人则担心它可能助长新的垃圾信息和机器人驱动的低质量内容。法律和发票复杂性也被提出作为担忧。
